# PCB 载板设计（立创 EDA）

阶段 5 实现。ESP32-S3 交互层载板，立创 EDA 工程文件放本目录（源文件 + 导出的 Gerber/BOM）。

## 设计需求（任务书要求）

1. **ESP32-S3 开发板**与 **TFT 液晶屏**通过**排针/排母**插装到 PCB（可拆卸）
2. 引出接口：
   - 舵机 ×N：3P 排针（PWM + 5V + GND），5V 走独立电源域
   - 麦克风：I2S（BCLK/WS/DIN）或模拟，按实际模块定
   - 扬声器：功放输入或直接喇叭端子（若板载功放如 MAX98357A）
   - 与 RDK X5 通路：UART 排针（3.3V 电平）+ 预留 USB
3. 供电：5V 输入端子；ESP32-S3 板自带 3.3V；**舵机 5V 与逻辑 5V 分离**，仅共地

## 工具链（老师推荐）

嘉立创EDA 扩展管理器装「**MCP Bridge**」+ VS Code/Cursor 装「**JLCEDA MCP Hub**」，
AI 可读原理图语义、审网表/DRC、引导选型放置（电源/地符号需手动加）。
全链路实战参考见 docs/references.md 的技术栈文章。

## 设计流程

1. 记录 ESP32-S3 开发板与 TFT 的实物引脚间距/排针定义 → 画封装
2. 原理图 → 引脚分配表（写入本 README，与 esp32/ 固件保持一致）
3. 布局：TFT 居板正面中央（做"脸"），排针插装高度考虑 3D 外壳
4. DRC 通过 → 导出 Gerber 下单
