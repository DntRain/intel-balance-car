## Arduino UNO 引脚分配

| UNO 引脚 | 连接 | 说明 |
|---|---|---|
| D2 (INT0) | 左编码器 A 相 | 外部中断计数 |
| D3 (INT1) | 右编码器 A 相 | 外部中断计数 |
| D4 | 左编码器 B 相 | 中断内读电平判向 |
| D5 | 右编码器 B 相 | 同上 |
| D6 | TB6612 STBY | 高电平使能（也可直接拉高省引脚） |
| D7 | TB6612 AIN1 | 左电机方向 |
| D8 | TB6612 AIN2 | 左电机方向 |
| D9 (PWM, Timer1) | TB6612 PWMA | 左电机 PWM |
| D10 (PWM, Timer1) | TB6612 PWMB | 右电机 PWM |
| D12 | TB6612 BIN1 | 右电机方向 |
| D13 | TB6612 BIN2 | 右电机方向 |
| A4 (SDA) | MPU6050 SDA | I2C |
| A5 (SCL) | MPU6050 SCL | I2C |
| A0 | 电池分压采样 | 分压比按电池节数定（如 3S 用 1/4 分压） |
| D0/D1 (RX/TX) | RDK X5 UART | ⚠️ 与 USB 烧录复用，烧录时断开；UNO TX→X5 RX 需分压到 3.3V |

不使用 MPU6050 的 INT 引脚（D2/D3 已被编码器占用），姿态解算用互补滤波轮询。

## 接线注意事项

1. **共地**：UNO、TB6612、RDK X5、ESP32-S3 所有 GND 必须连通
2. **电平**：UNO 是 5V 系统，RDK X5 / ESP32-S3 是 3.3V —— 所有 UNO TX 出去的信号都要分压或电平转换
3. **舵机独立供电**：舵机 5V 单独一路 DC-DC，只共地，防止堵转电流拉崩逻辑电源
4. **RDK X5 供电**：峰值电流大，DC-DC 至少 5V/3A，线径要够
5. **MPU6050 安装**：贴近轮轴轴线、远离电机（磁干扰）；安装方向记录到固件常量
6. **编码器电源**：霍尔编码器接 5V（UNO 侧），A/B 相信号 5V 电平直接进 UNO 无需转换

供电拓扑见 [architecture.md](architecture.md)。
