# 3D 建模与固定件（Fusion 360）

阶段 6 实现。Fusion 源文件（.f3d）与打印用 STL/3MF 放本目录。

工具链（老师推荐，二选一，详见 docs/references.md）：

- **openclaw-fusion**：AuraFriday MCP-Link + Fusion MCP Add-in 直控 Fusion，截图闭环验证

## 固定件清单（任务书要求）

| 零件 | 安装对象 | 设计要点 |
|---|---|---|
| RDK X5 支架 | 平衡车中层 | 留散热风道与接口出线空间；X5 较重，靠近整车重心 |
| PCB 支架 | 车体上部/前部 | TFT 屏做"脸"朝前；排针插拔方向留空间 |
| 雷达支架 | 车体顶部 | 360° 无遮挡、尽量水平、高度避开人腿以下杂物 |
| 麦克风固定件 | 靠上、远离电机 | 减少电机噪声拾入 |
| 扬声器固定件 | 车体前/侧 | 出音孔朝外 |
| 舵机固定件 ×N | 头部/手臂 | 与 PCB 舵机接口线长匹配 |

## 布局原则

- **重心居中偏低**：电池最底层，RDK X5 中层，轻的（PCB/屏/雷达）在上
- 所有零件先量实物尺寸（或找官方 step 模型），装配体中检查干涉
- 先测量 → 参数化建模 → 打印验证单件 → 总装
